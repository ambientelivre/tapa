The plugin

Tapa was developed by Oncase team to achieve an internal automation demand and it is distributed .

The purpose

The purpose behind this app is to be able to:

  • Change the appearance of Pentaho's Login Page - in an elegant fashion;
  • Structure a way to interchange templates and assets with the minimum impact to the platform;
  • Do dynamic css/js includes to pentaho app contexts;
